Este projeto consiste no desenvolvimento de um Componente de Interface (UI Component) estático, simulando o "Card de Filme/Série" encontrado em plataformas de streaming populares.

Criado como exercício prático do **1º Trimestre** da graduação em Engenharia de Software no **Instituto Infnet**, o projeto foca em demonstrar a capacidade de construir layouts complexos com alta fidelidade visual, desde as etapas iniciais do curso.

##  Destaques Técnicos

O desenvolvimento priorizou a organização e a precisão na replicação de um design de interface funcional:

* Componentização: Criação de um módulo (o card) que pode ser reutilizado em diferentes contextos de uma aplicação.
* Layout Detalhado: Uso de CSS para estruturar camadas complexas (Header com ícones, corpo de informações e footer de ações) dentro do card principal.
* Integração de Recursos: Utilização do Font Awesome para incorporar ícones de forma eficiente (menu, play, reações).
